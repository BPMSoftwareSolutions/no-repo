# Runtime Data-Access Contracts

This workspace contains the generated runtime contract examples and binding registries for the refactored self-learning, code-file-inventory, and workflow-store surfaces.

Files:

- `self_learning_store.contracts.json`
- `self_learning_store.bindings.json`
- `code_file_inventory.contracts.json`
- `code_file_inventory.bindings.json`
- `workflow_store_implementation_packet_store.contracts.json`
- `workflow_store_implementation_packet_store.bindings.json`
- `workflow_store_implementation_governance_store.contracts.json`
- `workflow_store_implementation_governance_store.bindings.json`
- `workflow_store_context_assembly.contracts.json`
- `workflow_store_context_assembly.bindings.json`

The contract JSON files define the canonical runtime keys.
The binding JSON files define the SQL resolver registry keyed by those contract keys.
